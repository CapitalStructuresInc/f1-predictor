Constructor Ranking Scoring

For each constructor: max(0, 5 minus the difference between your predicted rank and actual rank). Exact position = 5 pts. Off by 1 = 4 pts. Off by 5 or more = 0 pts.

Race Podium Scoring

Correct P1 = 3 pts. Correct P2 = 2 pts. Correct P3 = 1 pt. If you pick a driver who finishes on the podium but in the wrong position, +1 bonus pt.

Over/Under Scoring

For numerical picks (Different Winners, Backmarker Best): closest to the actual number wins the full points. If equidistant, earlier submission wins.

Tiebreaker

If tied at season end: (1) most correct Big Picture picks; (2) most correct constructor positions; (3) most race podium points.
